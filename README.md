Style Board is a Django web application that allows users to plan outfits from Monday–Sunday, upload inspiration images, add notes, and optionally share outfits publicly.

## Features
- User authentication (signup/login/logout)
- Create, edit, and delete outfit plans
- Weekly outfit planning (Monday–Sunday)
- Upload images for clothing, shoes, makeup, and accessories
- Add store links (Macy’s, H&M, etc.)
- Public outfit feed
- Admin moderation tools
